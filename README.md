## picturemyworld-server

Framework: Node.js

### How to start the server

Execute:

```
npm install
npm run createDB
npm start
```

### Postgres

Open Postgres in terminal:
```
psql -U postgres
```

Connect to database:
```
\c picturemyworld;
```

To list all databases:
```
\list
```

To list all tables:
```
\dt
```
